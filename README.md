# 🛒 Loja Virtual

Projeto de e-commerce desenvolvido como **teste técnico para vaga de desenvolvedor front-end**, utilizando React + Vite.

## 🚀 Funcionalidades

- Listagem de produtos
- Busca por nome
- Filtro por categoria
- Paginação client-side
- Carrinho de compras com controle de quantidade
- Checkout com toast de sucesso
- Produtos relacionados

## 🧠 Tecnologias

- React
- Vite
- Context API
- Tailwind CSS

## ⚙️ Como rodar

```bash
# Rodar frontend (http://localhost:5173)
npm install
npm run dev

# Rodar API fake (http://localhost:3001/products)
npx json-server dbTeste.json --port 3001
```
